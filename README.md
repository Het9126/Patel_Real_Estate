# Patrl Real Estate

This is a React/Express full-stack application.

## Common commands

- `npm run dev` - start development server
- `npm run build` - build for production
- `npm run start` - run built server
- `npm run check` - run TypeScript compiler
- `npm run db:push` - run drizzle migrations
- `npm run update-admin -- <username> <password>` - change or create the admin account (password will be hashed with bcrypt)

Example:

```powershell
npm run update-admin -- admin securePass123
```
